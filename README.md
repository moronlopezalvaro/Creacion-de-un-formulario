# Creación de un formulario
Estructura del formulario: Diseñar una estructura funcional con form y determinar el mejor método de envío (POST o GET), y un action simulado (por ejemplo, #).

# Campos básicos y avanzados:
Incluir una sección de información personal que recoja el nombre, correo y teléfono.

Añadir campos de dirección.

Incluir campos adicionales.

# Interactividad y experiencia de usuario:
Añadir campos de selección múltiple para intereses, y usar selectores input type="checkbox" y input type="radio" según el tipo de pregunta.

Permitir personalización de opciones en campos como "áreas de interés", con un input type="text" que permita ingresar una opción personalizada si ninguna es relevante.

# Estilo y Accesibilidad:
Agregar etiquetas label correctamente vinculadas a cada campo.

Explorar el uso de roles de accesibilidad para hacer el formulario más inclusivo.

Añadir placeholder y title en los campos para mayor claridad.
